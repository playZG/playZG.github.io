PHP基础知识

运算符

运算符：运算符用于执行程序代码运算，运算符大致可以分为5种类型：算术运算符、连接运算符、关系运算符、赋值运算符、逻辑运算符、比较运算符、位运算符

运算：有一个值或多个值经过某种变化(运算符)得到新值的过程

按操作数分类：

   一元运算符/单目运算：$var

   二元运算符/双目运算：$a=$b

   三元运算符/三目运算：(expr1) ? (expr2) : (expr3) 对 expr1 求值为 TRUE 时的值为 expr2，在 expr1 求值为 FALSE 时的值为 expr3。

算术运算符：

运算符：+、－、*、/、%、++、–

在除法和求余中被除数不能为０

变量＋＋：如果变量使用该方式进行自增运算的同时进行其他运算或者操作，先使用变量然后在进行＋1操作，然后再进行其他操作

＋＋变量：如果变量使用该方式进行自增运算的同时进行其他运算或者操作，先使用变量＋1操作然后在进行其他操作

变量－－：如果变量使用该方式自减运算的同时进行其他操作，先使用变量然后在－1运算

－－变量：如果变量使用该方式自减运算的同时进行其他操作，先进行变量－1操作，然后在进行其他操作

特点：

１．无论是自增运算还是自减运算都是在原数值基础上进行＋1或者－1操作得到新值

２．自增运算或者自减运算，不会改变原有的数据类型

３．　不要对布尔类型进行自增或自减操作

字符串运算符：

   拼接任意字符串的，任意的标量类型和字符串进行连接结果都为字符串类型

赋值运算符：

赋值运算符：＝、＋＝、－＝、*=、/＝、％＝、.＝

$one+=$two相当于$one=$one+$two

$one*=$two相当于$one=$one*$two

注意：在赋值运算符的左侧必须是变量，右侧是一则表达式

比较运算符：

比较运算符：>、<、>=、<=、==、!=、===、!＝

逻辑运算符

逻辑运算符：&&、and　操作数都为真结果才为真其余全是假，|| 、or　操作数有一个为真结果即为真，其余全是假，！真变假 假变真，xor相同为真 不同为假

其他运算符

@错误抑制符　作用：屏蔽错误，只能屏蔽单行错误，尽量少用@，效率低

系统命令执行符：｀｀，shell_exex()

数组元素访问符：＝>

对象成员访问符：－>

在运算符中如果涉及到优先级问题　建议使用()

流程控制

流程控制：对程序执行流程的控制就是流程控制

流程：程序执行的顺序就是流程

流程的分类：顺序结构、分支结构(选择结构)、循环结构

顺序结构：程序的自上而下的执行顺序就是顺序结构，所有的程序默认都是顺序结构

分支结构(选择结构)：程序在执行工程中经常会做出选择，忽略一定的代码执行，这种执行结构称之为分支结构(选择结构)

分支结构

分支结构的分类：单向分支、双向分支、多向分支、巢状分支

单项分支结构：

１．格式：

if(条件表达式)

    1条PHP语句

如果表达式为真则会执行表达式之后的一条PHP语句，如果表达式为假，那么表达式之后的一条PHP语句不会执行。条件表达式的结构只能控制PHP之后的一条PHP代码

２．格式：

if(条件表达式){

&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;&ensp;PHP代码

       }

如果表达式为真则会执行{}之内的PHP语句块，如果表达式为假那么表达式之后的{}之内的PHP语句不会被执行

双向分支：

１．格式：

if(条件表达式)

    1条PHP语句

else

   １条PHP语句

如果表达式为真则会执行表达式之后的一条PHP语句，如果表达式为假则会执行else之后的一条PHP语句

注意：双向分支if后只能有一条PHP语句，否则报语法错误

２．格式：

if(条件表达式){

   多条PHP语句

}else{

   多条PHP语句

}

如果表达式为真则会执行if之后的语句块，否则会执行else之后的语句块

１．格式：

３．如果想实现分支结构，需要分支代码结束之后加上一个break

５．如果switch…case的分支都没有匹配到，那么会走默认分支default

循环结构

循环：按照规定反复执行某些操作的过程就是循环

特点：在计算机涉及到算法中，最能体现计算机特长就是循环，能够节省程序代码，按照一定的条件反复执行程序语句块，直到条件为假不成立为止，习惯上把条件称为循环条件，反复执行的程序称之为循环体

循环结构：while型循环、do…while型循环、for型循环、foreach循环

while型循环：常用于未知次数循环

格式：

while(条件表达式){

   循环体/语句块

   自增/自减

}

do…while型循环：

格式：

do&#123;
    循环体/语句块
    自增/自减
&#125;while(表达式)；

特点：先执行一遍循环体，在判断是否继续执行下一次的循环，如果开始条件是合法的则会执行循环体内容，知道条件不成立为止，如果条件不成立则会执行一遍循环体内容

while和do…while区别

只有do…while循环输出了，while循环什么都没有输出

for型循环：主要用于已知次数的循环

格式：

for(初始化变量；条件表达式；自增/自减条件)&#123;
    循环体语句块
&#125;

break：在分支结构中跳出当前分支，在循环结构中退出当前循环

continue：继续过滤特殊数据，在循环中能够跳出某一次循环开始下次新的循环，continue在for循环中直接跳过当前操作然后满足条件则继续执行，直到条件不成立退出循环，在while循环中如果不先进行自增或自减操作都会报错或者发生死循环

break和continue的区别

exit()退出当前脚本终止脚本执行

函数

函数：提高利用率，减少开发时间，函数就是一个功能

函数的分类：

１．根据是否为系统函数：自定义函数，系统函数

２．根据是否具有返回值：返回值函数，执行过程函数

３．根据函数结构：一般函数，变量函数，回调函数，内部函数，递归函数

函数的基本语法：

   基本格式：

function 函数名()&#123;
    函数体php语句
&#125;

注意：

１．函数都要以关键字function开始

２．函数名后面要紧跟小括号

３．小括号下面跟大括号，小括号里面是函数体

４．函数调用需要函数加上小括号

变量函数：变量函数就是使用一个变量来设置值为函数名称的字符串，在调用函数的时候不在使用函数名()而是使用变量()的形式来调用函数

回调函数：就是变量函数的实际应用

内部函数：

１．在函数外部调用内部函数时需要先调用外部函数再次调用内部函数否则报错

２．在函数内部调用内部函数时需要在内部函数定义之后再调用，否则找不到函数

递归函数：实际上就是自己调用自己

我们写一个递归函数，功能是统计目录的大小

函数的相关特点：

1.函数在定义之后，不调用不执行，函数的调用直接使用函数名()

2.函数调用不分先后顺序，可以在函数定义之前调用，也可以在函数定义之后调用，和程序解析流程有关系

函数名的命名规范：

   １．可以使用英文，不可以使用中文

   ２．可以使用数字，但不能以数字开头

   ３．不可以使用特殊符号，下划线除外

   ４．遵循驼峰命名，函数的命名要有意义

   ５．函数不区分大小写(系统函数、自定义函数)

   ６．函数的命名不能和已存在的函数名同名(自定义函数、系统函数)，否则报错

关于函数返回值问题

具有return和不具有return的区别

１．具有return的函数称之为返回值函数，可以被程序接收

２．不具有return的函数称之为执行过程函数，不能被程序变量接收

在函数中return不但可以返回结果还可以终止程序的执行，函数执行的过程中一旦遇到return就终止结束了，一般情况下，一个程序只能有一个return,但是如果函数体中存在着分支结构那么每个分支都可以具有一个return

函数的参数问题

在函数的语法中，括号中的内容就是参数，函数定义部分的参数就是形参(形式上的参数)，调用函数时传入的参数称之为实参(实际参数)

１．函数如果有形参，调用时必须传入实参

２．函数形象和实参必须－－对应(在没有默认值的情况下)

３．默认值在形参处定义变量赋值

４．如果同时具有形参和实参的情况下，最终结果以实参为主

５．如何强制一个参数必须填写(在没有默认值的情况下)

６．在传入实参的时候需要注意，实参和形参要做好对应，放置逻辑出错

７．将实参赋值为形参的过程就是跟简单的变量赋值是一样的

作用域

作用域定义：就是一个变量的作用范围，或者叫做生命周期

内部变量：就是在函数体内声明的变量，内部变量的作用域只在函数体内生效，程序执行完以后自动销毁(垃圾回收机制)

外部变量：就是在函数体外声明的变量，不能在函数体内使用

超全局变量：外部变量和内部变量是同一个变量。例：$_GET、$_POST、$_COOKIE、$_SESSION、$_GLOBALS

正常情况下函数内部变量和外部变量是互不干扰的

静态变量：这个变量只会初始化一次，在运行的时候它会记录上一次的值，static变量不会销毁

还有一种情况，懒得打字上图

匿名函数：所谓的匿名函数就是没有名字，如果想调用，赋值给一个变量，变量加上()跟普通函数调用一样

变量的全局化方式

１．使用关键字global声明变量

２．使用$GLOBALS超全局数组定义变量$GLOBA[　‘外部变量名称’　]

文件包含

PHP提供了两个非常简单却非常有用的语句，include()或者require()语句，可以将一个文件载入到PHP脚本中。

include:不是函数而是一种语言结构，书写方式２种

   include　‘文件路径’

   include(文件路径)

require:不是函数而是一种语言结构，书写方式２种

   require 　‘文件路径’

   require(文件路径)

include和require的区别：

错误产生后页面的执行方式：

include　一个不存在的文件提示的是warning级别错误，但是程序会继续向下执行

require　一个不存在的文件提示的是error级别错误，会终止程序的执行

include和include_once的区别：

include()　２次相同的文件，如果文件中存在函数会报函数重名错误

include_once()　在包含多次相同文件的情况下不会报错

正常情况下推荐使用include，因为include效率高，而include_once()需要判断效率低

正则表达式

基础知识：正则表达式是一种描述一段文本模式的方法。正则表达式就是一种特殊的字符串，只是字符串中有一些语法规则还有特殊符号，正则表达式必须配合正则函数一起使用否则就是一个普通的字符串

１．字符集和类

使用字符集可以马上给出比精确匹配功能还要强大的正则表达式。字符集可以用于匹配属于特定类型的任何字符，事实上它是一种通配符。

例如：.at

它可以是”cat“、“sat”、“mat”等进行匹配，通常这种通配符匹配操作系统中的文件名。

例如：[a-z]at

限定它是a到z之间的字符，任何包含在方括号( [ 　] )中的内容都是一个字符类，一个被匹配字符所属的字符集合，注意：方括号中的表达式只匹配一个字符。

例如：[^a-z]

可以用来匹配任何不在a和z之间的字符，当把脱字符号(^)包括在方括号里面时，表示否

用于POSIX风格的正则表达式的字符类

类 匹配 类 匹配
[[:alnum:]] 文字数字字符 [[:punct:]] 标点符号
[[:alpha:]] 字母字符 [[:blank:]] 制表符和空格
[[:lower:]] 小写字母 [[:space:]] 空白字符
[[:upper:]] 大写字母 [[:cntrl:]] 控制符
[[:digit:]] 小数 [[:print:]] 所有可打印的字符
[[:xdigit:]] 十六进制数字 [[:graph:]] 除空格外所有可打印的字符

２．重复

通常我们会希望某个字符串或字符类将不止一次地出现。可以在正则表达式中使用两个特殊字符代替，符号“*”表示这个模式可以被重复0次或更多次，符号“＋”则表示这个模式可以被重复1次或更多次.这两个字符应该放在要作用的表达式的后面

例如：[[:alnum:]]+

表示“至少有一个字母字符”。

３．子表达式

通常将一个表达式分隔为几个子表达式是非常有用的，例如：“至少这些字符串中的一个需要精确匹配”。可以使用圆括号来实现，与在数学表达式中的方法一样

例如：(very*)large

可以匹配“large”、“very large”、“very very large”等

４．子表达式计数

可以用在花括号｛｝中的数字表达式来指定内容允许重复的次数。　可以指定一个确切的重复次数｛３｝表示重复３次，或者一个重复次数的范围｛２，４｝表示重复２~4次。

例如：(very)｛1,3｝

表示匹配“very”、”very very”和“very very very”

５．定位到字符串的开始或末尾

当脱字符号(^)用于正则表达式的开始，表示子字符串必须出现在被搜索字符串的开始处，字符“$”用于正则表达式的末尾，表示子字符串必须出现在字符串的末尾。

例如：^play

以下是在字符串开始处匹配play

例如：play$

匹配play出现在字符串末尾处的字符串

例如：^[a-z]$

这个模式将匹配只包含a到z之间一个字符的字符串

６．分支

可以使用正则表达式中的一条竖线来表示一个选择。

如果要匹配com、edu、d或者net

例如：com|edu|net

７．正则表达式函数

preg_match() 执行一个正则表达式匹配

preg_match_all() 执行一个全局的正则表达式匹配

笔记

提示：这部分是笔记，感觉这部分更实用一点，上面那部分是按照书上写的

正则表达式可以分为４个部分：１正则定界符、２正则原子、３．正则元字符、４．正则模式修正符

一个最简单的正则表达式必须包含２部分：定界符、原子

一、

在PHP中的定界符除了0-9a-zA-Z不能用其他的都可以用做定界符，定界符都是成对出现的

二、

正则原子：要找的或者是要匹配的字符，要写在定界符中

原子是正则表达式最基本的组成单位，每个正则表达式中最少含有一个原子

匹配所有的数字、字母、空白特殊符号
\d 代表任意一个数字
\D 代表任意一个非数字
\w 代表任意一个字
\W 代表任意一个非字
\s 代表任意一个空白
\S 代表任意一个非空白

三、

元字符：用来修饰原子的
*　用来修饰原子出现的次数０次或多次
+　用来修饰原子出现次数一次或多次
？ 用来修饰原子出现次数０次或１次
{m}　用来修饰原子只能出现m次
{m,n} 用来修饰原子出现的次数只能出现m次到n次(包含m次和n次)
{m,}　用来修饰原子出现的次数最少为m次
｜　用来修饰原子匹配左右两端只要出现一个即可
^或\A　用来修饰原子必须以什么开头，必须要写在要修饰原子前
$或\Z　用来修饰原子必须以什么结尾，必须要写在要修饰原子的后面

()的作用：

１．改变优先级

２．子模式，整个表达式是一个大模式，小括号将原子变成每个独立的子模式(取消子模式(?:)

３．反向作用

四、

模式修正符：用来修饰整个正则表达式，可以修正整个正则表达式的解释，或者扩充整个正则表达式的功能

特点：

１．模式修正符，要写在定界符之外

２．一个模式修正符就是一个功能可以组合使用

i　用来修饰整个正则表达式不区分大小写
m　用来修饰整个正则表达式可以视为多行，默认情况下是一行
s　用来修饰正则表达式中的.可以匹配换行符，默认.是不匹配换行符的
x　用来修饰正则表达式中的空白，设置x表示不允许出现空白(换行、缩进、空格)
U　表示拒绝贪婪模式，默认进行贪婪模式匹配，不能和.?*或者.+?同时使用

文件上传

通过PHP,可以把文件上传到服务器

前端的准备：

１．表单提交方式必须为post

２．而且form表单属性enctype必不可少属性值为multipart/form-data,在表单属性需要二进制时，比如文件内容时会设置这个属性

３．需要在上传表单中设置一个隐藏域，设置上传文件大小，name=MAX_FILE_SIZE ,value=固定的自定义的允许表单上传文件最大值

后台部分：

保证配置文件中的file_upload=on处于开启状态、upload_max_filesize = 2M 上传文件的大小限制、post_max_size = 8M 表单post允许传递的最大值限制、memory_limit = 128M 最大的使用内存大小

做文件上传不在使用post来接受表单数据，而是使用$_FILES来接收表单数据

预定义变量$_FILES数组的５个内容

$_FILES[‘file’][‘name’]——客户端机器文件的原名称
$_FILES[‘file’][‘type’]——文件的 MIME 类型
$_FILES[‘file’][‘size’]——已上传文件的大小，单位为字节
$_FILES[‘file’][‘tmp_name’]——文件被上传后在服务端储存的临时文件名
$_FILES[‘file’][‘error’]——和该文件上传相关的错误代码

其中$_FILES[‘file’][‘error’]的可以有下列取值和意义：

０－没有错误发生，上传成功

１－上传的文件超过了php.ini中upload_max_filesize选项限制的值

２－上传文件的大小超过了HTML表单中MAX_FILE_SIZE选项指定的值

３－文件只有部分被上传

４－没有文件被上传

６－找不到临时文件

７－文件写入失败
