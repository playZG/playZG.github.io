PHP介绍

PHP即超文本预处理器，是一种通用开源脚本语言。PHP是在服务端执行的脚本语言，主要用于Web开发领域。它能够收集表单数据(处理数据能力）、对文件　http字符串处理、动态输出图像、服务器端操作、编写数据库支持的网页、会话跟踪处理。

常见的网站架构

WAMPwindow+apache+mysql+php
LAMPlinux+apache+mysql+php
LNMPlinux+nginx+mysql+php
J2EEunix+tomcat(weblogic)_oracle(mysql)+jsp
ASP.NETwindows+iis+sqlserver+asp

建议大家安装一个集成环境去学习。

新建文件命名：

１．PHP文件命名要有意义

２．不能使用中文，可以使用英文字母和数字

３．不能使用特殊符号下划线除外

４．文件后缀以.php

５．PHP文件可以不包含php代码

PHP使用方式

１．以\<?php开始　　php代码　　以?\>结束(完整风格，经典风格)。

２．以\<%开始　　php代码　　%\>结束，短风格模式，默认不是开启的，需要手动开启，在php.ini(php的配置文件中)找到short_open_tag将off改成on即可开启。

３．以\<％开始　　php代码　　%\>结束，asp风格模式，默认也是关闭的需要手动开启，去php.ini(php的配置文件中)找到asp_tags将off改成on即可开启。

４．以\<script language="php"\>开始　　php代码　　以\<script\>结束　　js风格，默认开启

    经典风格推荐使用，不收配置文件控制，都是自动开启，所有的PHP程序都支持，都使用该方式，不存在兼容性问题。

PHP的其他相关特性

１．PHP是一门脚本语言，可以和其他语言进行混合编写，html、css、javascript

２．PHP结束分隔符(结束指令分隔符)

    任何的PHP语句之后都要有一个分号结束。

    语法定界符?>也有结束指令分隔的作用

程序注释

PHP注释：单行注释//、＃　多行注释/**/

注释的作用：区域介绍说明，代码标注，后期维护，错误调试(程序调试)

变量

PHP中的变量用一个美元符号后面跟变量名来表示。变量名是区分大小写的。简单的说变量就是变化的量。

变量的命名规则：

１．变量的声明必须以$开始

２．不可以使用中文，必须使用英文名

３．可以使用数字，但是不能以数字开头

４．不能使用特殊符号(_除外)

５．变量命名严格区分大小写

６．严格遵循驼峰命名法

普通变量的赋值方式(取值的过程)：

格式：变量２＝变量１

特点：两个变量互不干扰，因为他们仅具有２个相同的值

变量的引用赋值：地址绑定

格式：变量２＝＆变量１

特点：２个变量互相干扰，因为他们具有相同的地址

可变变量：可以同时使用多个$，当有多个$时，解析时是从右向左以此解析，直到解析到最后一个$为止

PHP中数据类型：三类八种

PHP中数据类型
标量类型整形浮点型字符串布尔类型
符合类型数组对象
特殊类型资源NULL

标量类型：

１．整形(int)：整形就是整数

    10进制、８进制、16进制

２．浮点型(float)：就是小数

    直接声明小数形式、科学计数法

３．字符串(string)

    使用双引号来定义字符串、使用单引号来定义字符串、定界符定义字符串

   在使用定界符声明字符串时需要注意：在变量之后使用<<<跟上自定义的标识符，通常都是大写的，写完之后必须换行，不能有任何的字符输出，如果想结束字符串，那么必须另起一行，而且要顶格写上自定义的标识符

区别　

   单引号只能转义少部分的转义字符，但是双引号能够转义大部分的转义字符。对变量的解析能力：单引号不解析变量，双引号解析变量，在双引号解析变量时一定要加上{变量名}

注意

    １．单引号不能嵌套单引号，双引号也不能嵌套双引号，但是单双引号可以互相嵌套

    ２．能够使用单引号绝对不使用双引号，效率不同

    ３．SQL语句全部使用双引号

    ４．定界符一般在定义文章或者复制文章时使用

４．布尔类型bool

    True、False，布尔类型常用于逻辑判断
