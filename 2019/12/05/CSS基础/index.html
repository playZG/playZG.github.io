## CSS概述

   CSS指层叠样式表

   CSS样式表极大地提高了工作效率

   作用：美化页面

   注释：/*注释内容*/

CSS使用

内联式或内嵌式

用法：在标签中使用style属性来引入对应的css属性

注意：style属性属于公有属性，任何标签都可以设置style属性

嵌入式

当页面当中借助style标签来引入css属性，可以放置在head头中也可以放置在body体中

导入式

在style标签中借助@import url(css 地址)从而来引入相应的css样式

外链式

借助link标签href属性用来引入外部css地址，rel属性引入外部的css文件和当前html页面的关系stylesheet还需要使用type来指定引入文件的类型text/css

取舍

１．内联式对单个标签有用，在设置js特效时经常使用

２．嵌入式可以作用于多个标签，在页面布局发生混乱时经常使用

３．导入式几乎不用

４．外链式几乎都在用

继承

我们定义了body的样式，在body中没有如果没有自己的样式，那么就会继承body的样式，如果有自己的样式，先按照自己的样式执行。

CSS选择器

派生选择器

通过依据元素在其位置的上下文关系来定义样式

通过li找到p元素并作用，css样式会有就近原则

id选择器

id选择器可以为标有id的HTML元素指定特定的样式，id选择器以 ‘#’来定义，id选择器可以和派生选择器一起结合使用.id选择器只能在文档中使用一次。

类选择器

类选择器是以一个点显示，也可以和派生选择器一起结合使用，类选择器能在文档中使用多次

属性选择器

对带有指定属性的HTML元素设置样式

CSS样式

背景

CSS允许应用纯色作为背景，也允许使用背景图像创建相当复杂的效

属性 描述
background-attachment 背景图像是否固定或者随着页面的其余部分滚动
background-color 设置元素的背景颜色
background-image 把图片设置为背景
background-position 设置背景图片的起始位置
background-repeat 设置背景图片是否及如何重复

CSS3背景

１．background-size：规定背景图片的尺寸

２．background-origin：规定背景图片的定位区域

３．background-clip：规定背景的绘制区域

文本

CSS文本属性可定义文本外观，可以改变文本的颜色、字符间距、对齐文本、装饰文本、对文本缩进

文本描述属性：text-decoration： overline上划线、underline下划线、line-through贯穿线、none取消下划线

行高设置：line-height，设置文本的垂直居中设置，只要保证行高和高度一致则垂直居中

文本的水平对齐方式：text-align： left左、center中、right右

字符间距：letter-spacing属性，属性值需要给尺寸单位、对英文字符和汉字均有效果

单词间距：word-spacing属性，属性值需要给尺寸单位，仅对英文单词有效果

首行缩进：text-indent： 属性值需要给尺寸单位

转换不同元素中的文本：text-transform属性值capitalize每个单词以大写字母开头、lowercase无大写，仅有小写字母、uppercase仅有大写字母

向文本添加阴影：text-shadow(４个参数,类似5px 5px 2px #FF00)

规定文本的换行规则：word-wrap　

文本溢出处理：

text-overflow:clib超过部分直接切割，ellipsis超过部分直接使用…来代替（用户体验更好）　文本溢出处理需要配合其他属性一起使用，１.设置文本在一行显示（不换行）hite-space属性值nowrap，超出部分处理设置overflow设置为隐藏hidden

字体

CSS字体属性定义文本的字体系列、大小、加粗、风格和变形

CSS3选择器

选择器

div > p{} 子元素选择器，匹配所有div元素的子元素p，p为div的一层后代

div + p{}　毗邻元素选择器，匹配紧随div元素之后的一个p元素

div ~ p{}同级元素通用选择器，匹配紧随div元素之后的同级p元素

属性选择器

div[title]{}匹配所有具有title属性的div元素，注意此处的div是可以省略的，匹配所有具有title属性的元素

div[title＝one]{}匹配具有title属性值为one的div元素，注意此处的div是可以省略的，匹配所有具有title属性值为one的元素

div[class~=one]匹配所有class属性具有多个空格分隔的值，其中一个值为one即可，注意div是可以省略的，匹配class属性中有one的元素

div[title|=hello]{}匹配的是所有具有title属性值具有多个连字符(-)分隔的值且第一个值为hello的div元素

div[title^=playzg]{}匹配title属性以playzg开头的

div[title$=playzg]{}匹配的具有title属性的div元素且最后的值为playzg

div[title*=playzg]匹配具有title属性的div元素有一个值为playzg即可

2D、3D

通过CSS3转换，我们能够对元素进行移动、缩放、转动、拉长或拉伸。转换是使元素改变形状、尺寸和位置的一种效果。可以使用2D、3D来转换元素

2D转换方法：

transform 属性向元素应用 2D 或 3D 转换。该属性允许我们对元素进行旋转、缩放、移动或倾斜。

2D

方法：

translate()：根据左(X轴)和顶部(Y轴)位置给定的参数，从当前元素位置移动。

rotate()：在一个给定度数顺时针旋转的元素。负值是允许的，这样是元素逆时针旋转。

scale()： 该元素增加或减少的大小，取决于宽度（X轴）和高度（Y轴）的参数

skew()：包含两个参数值，分别表示X轴和Y轴倾斜的角度，如果第二个参数为空，则默认为0，参数为负表示向相反方向倾斜。

matrix()：matrix 方法有六个参数，包含旋转，缩放，移动（平移）和倾斜功能。

有兴趣可以自己试一下

3D

方法：

rotateX()：围绕其在一个给定度数X轴旋转的元素。

rotateY()：围绕其在一个给定度数Y轴旋转的元素。
