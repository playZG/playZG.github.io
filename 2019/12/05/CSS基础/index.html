CSS概述

   CSS指层叠样式表

   CSS样式表极大地提高了工作效率

   作用：美化页面

   注释：/*注释内容*/

CSS使用

内联式或内嵌式

用法：在标签中使用style属性来引入对应的css属性

注意：style属性属于公有属性，任何标签都可以设置style属性

嵌入式

当页面当中借助style标签来引入css属性，可以放置在head头中也可以放置在body体中

导入式

在style标签中借助@import url(css 地址)从而来引入相应的css样式

外链式

借助link标签href属性用来引入外部css地址，rel属性引入外部的css文件和当前html页面的关系stylesheet还需要使用type来指定引入文件的类型text/css

取舍

１．内联式对单个标签有用，在设置js特效时经常使用

２．嵌入式可以作用于多个标签，在页面布局发生混乱时经常使用

３．导入式几乎不用

４．外链式几乎都在用

CSS基础语法

属性选择器｛

               属性：属性值

            ｝

属性大于１个之后，属性之间用分号隔开

如果值大于１个单词，则需要加上引号

选择器分组

给多个元素加上同一个样式

h1,h2,h3,h4,h5{color:red}

继承

我们定义了body的样式，在body中没有如果没有自己的样式，那么就会继承body的样式，如果有自己的样式，先按照自己的样式执行。

CSS选择器

派生选择器

通过依据元素在其位置的上下文关系来定义样式

通过li找到p元素并作用，css样式会有就近原则

id选择器

id选择器可以为标有id的HTML元素指定特定的样式，id选择器以 ‘#’来定义，id选择器可以和派生选择器一起结合使用

类选择器

类选择器是以一个点显示，也可以和派生选择器一起结合使用

属性选择器

对带有指定属性的HTML元素设置样式

CSS样式

背景

CSS允许应用纯色作为背景，也允许使用背景图像创建相当复杂的效

属性 描述
background-attachment 背景图像是否固定或者随着页面的其余部分滚动
background-color 设置元素的背景颜色
background-image 把图片设置为背景
background-position 设置背景图片的起始位置
background-repeat 设置背景图片是否及如何重复

CSS3背景

１．background-size：规定背景图片的尺寸

２．background-origin：规定背景图片的定位区域

３．background-clip：规定背景的绘制区域
