PHP面向对象

面向对象的特征：

三大特征：封装、继承、多态

封装性：解决逻辑上的bug

封装的三个关键字：private、protected、public

关键字在类的内部访问属性在类的外部访问属性在类的内部访问方法在类的外部访问方法
public可以可以可以可以
private可以不可以可以不可以
protected可以不可以可以不可以

可以看到只有public定义的方法say()能访问，protected定义的he()方法和public定义的eat()方法在类的外部并不能访问

 

private和protected的特性：

​ 1.设置为private和protected关键字的成员都不能在类的外部进行调节

​ 2.私有化成员在类的内部可以进行调用使用

​ 3.一般情况下不要对构造方法进行私有化设置

PHP常见的魔术方法：

__get()

​ 触发时机：在类的外部访问私有化或者受保护成员时自动触发

​ 作用：返回私有化成员属性的值

​ 参数：1个 自动传入私有化成员属性名称的字符串

​ 返回值：需要返回值

protected定义的属性和public定义的属性通过__get()可以访问

__set()

​ 触发时机：在类的外部对私有成员属性进行设置时自动触发

​ 作用：为类能够在类的内部对私有化成员进行属性的的修改

​ 参数：2个 1个自动传入私有化成员属性名称的字符串 2个修改的值

​ 返回值：通常没有

__isset()

​ 触发时机：在类的外部对私有化成员属性isset检测时自动触发

​ 作用：代替外部的isset对私有化成员设置值检测

​ 参数：1个 自动传入私有化成员属性名称的字符串

​ 返回值：需要，通常是布尔值

__unset()

​ 触发时机：在类的外部对私有化成员属性unset摧毁或删除是自动触发

​ 作用: 代替外部的isset对私有化成员设置值检测

​ 参数：1个 自动传入私有化成员属性名称的字符串

​ 返回值：需要，通常是布尔值

__autoload()

​ 触发时机：实例化对象时，或者使用的类需要继承其他类，并且在当面页面不存在要实例化或者要继承的那个类时自动触发

​ 作用：实现类的自动加载

​ 参数：1个 自动传入的类名

​ 返回值：无

__toString()

​ 触发时机：在echo一个对象时自动触发，或者将对象以字符串形式输出时触发

​ 作用：防止报错

​ 参数：无

​ 返回值：必须有而且是一个字符串

__clone()

​ 触发时机：在对一个对象clone时触发

​ 作用：在克隆对象时同时修改部分成员属性

​ 参数：无

​ 返回值：无

总结：

​ 只有对私有化或受保护的成员属性操作时才会触发魔术方法

​ 所有的魔术方法都是自动触发，根据功能不同所以参数也不相同，返回值也不相同

​ 魔术方法在使用时需要进行逻辑判断

继承：

​ 父类：被其他类继承的那个类叫做父类，也叫做基类

​ 子类：继承其他类的类叫做子类，也叫派生类

继承格式：

​ class 子类 extends 父类{

​ 子类特有的成员属性

​ 子类特有的成员方法

}

继承特性;

​ 1.子类继承父类，既具备父类的成员属性也具备父类的成员方法

​ 2.子类可以具有自己独特的成员属性和成员方法

​ 3.子类可以覆盖继承而来的父类方法，只需要在子类中重新定义一次即可（这种方式称为重载）

​ 4.一般在重载之后还想调用父类的方法parent::父类方法即可

​ 5.构造方法也可以继承只需要在子类中parent::__construct（），不要忘记传入父类构造方法中的参数

继承特点：PHP是单继承，也就是多个子类只有一个父类

继承的额外规则：子类的成员封装关键字要大于等于父类关键字

 

final 关键字：

​ 如果对父类中的方法使用final在子类中无法重载改方法（禁止重载）

​  当我们在子类中重载用final关键字处理的gpx()方法时，会发现程序报错

​  然后把final关键字去掉，再重载gpx()方法

​ 对类使用final关键字结果是该类无法被继承（禁止继承）

​  继承用final关键字处理的LiuBei类时，程序会报错

​  把final关键字去掉，再继承LiuBei类

const 关键字（定义类中常亮）：

​ 访问方式：

​ 类外部：类名::常量名

​ 类内部：类名::常量名 self::常量名

三个关键字：

​ $this 表示的是当前对象 存在new

​ parent 表示的是父类

​ self 表示的是当前类

static关键字：声明静态成员

​ 作用：在某些情况下能过在不存在对象的时候就可以访问成员，这个时候就需要使用static来声明静态使用

静态成员的访问方式：

​ 成员属性

​    类内部：

​       类名::$成员属性名

​       self::$成员属性名

​   类外部：

​      类名::$成员属性名

成员方法：

​   类内部：

​      类名::方法名（)

​      self::方法名（）

​    类外部：

     类名::方法名（）

特点：

​ 1.所有的成员都可以声明为静态的

​ 2.如果想声明静态方法，那么必须确保该方法下没有动态成员（$this）

​ 3.程序在读取类的时候如果该方法中不存在$this，那么默认这个方法就是静态的

​ 4.静态成员的访问速度远高于动态成员的访问速度
